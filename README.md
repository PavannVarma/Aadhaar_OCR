# Aadhaar OCR Project

This project uses Java and Tesseract OCR to extract metadata from Aadhaar card images.

## How to Run

1. Clone the repo
2. Place `Telugu.traineddata` in the `tessdata/` folder
3. Compile and run the Java code inside the `src/` directory

## Dependencies

- Tesseract OCR
- Tess4J
